User prefers direct, concise responses in Chinese. Skip preamble, go straight to the point.

**Why:** Efficiency — user is experienced and doesn't need hand-holding.

**How to apply:** Keep answers short. Use agents (Sonnet) for bulk work like tests, docs. Show options/tradeoffs when there are real choices, but don't over-explain obvious things. User will say "好" when satisfied and move on.
